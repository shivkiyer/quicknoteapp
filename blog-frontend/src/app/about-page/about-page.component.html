About QuickNote


This app was built to create a searchable database for developers as a knowledge resource while the develop their projects or learn new technology. We developers gather knowledge from a wide variety of sources such a official guides, blogs, Stack Overflow and even social media sites as we work our way through a project.

This app lets you create "Topics" which could be projects and within those "Subtopics" which are basically different aspects of the project and finally within these "Notes". A note is a collection of text and code however you decide to mix them. All this is written to a MongoDB database. So the end result is a searchable database that can be queried when you encounter a similar problem again.

And as final addition, there is the collaborative learning aspect. Developers can share projects and write together as they work on different aspects of a project. Projects can be assigned permissions of different levels and projects can be private or can be read by all.
